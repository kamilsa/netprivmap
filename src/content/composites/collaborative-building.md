## Description

A modular block-building pipeline that leverages TEEs and anonymous broadcast to allow multiple actors to collaboratively produce blocks without centralized gateways.

## Key properties

Censorship resistance, metadata privacy, permissionless building, cross-domain MEV handling.

## Limitations

Reliance on hardware trust roots (TEEs), high coordination complexity between co-builders.
